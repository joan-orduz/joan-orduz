## 🚀 Featured Project

### 📚 Comics Marketplace API

[![Repo](https://img.shields.io/badge/View%20Repository-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/joan-orduz/comics-marketplace-api)
![TypeScript](https://img.shields.io/badge/TypeScript-98.9%25-3178C6?style=flat-square&logo=typescript)
![Stars](https://img.shields.io/github/stars/joan-orduz/comics-marketplace-api?style=flat-square&color=yellow)

</div>

> Full REST API for a comics buying and selling marketplace, with real payment processing via **PayU**, built with production-grade architecture.

**✅ Key Features:**

| Module | Description |
|--------|-------------|
| 🔐 **Auth & Roles** | JWT system with `ADMIN`, `SELLER`, `USER` roles and global guards |
| 📖 **Comics Catalog** | Full CRUD with filters, search, and pagination |
| 💳 **PayU Payments** | Complete integration: checkout, secure webhook, and MD5 validation |
| 🔒 **Atomic Transactions** | Pessimistic locks (`SELECT FOR UPDATE`) to prevent race conditions |
| ⚡ **Distributed Cache** | Redis with `cache-manager` for query optimization |
| 📋 **Job Queue** | Bull for asynchronous task processing |
| 🛡️ **Rate Limiting** | Global throttling for abuse protection |
| 📄 **Swagger Docs** | Fully documented and interactive API |

**🧰 Stack:**
`NestJS 11` · `TypeScript` · `PostgreSQL 14` · `Redis 7` · `TypeORM` · `Bull` · `JWT + Passport` · `PayU` · `Docker` · `Jest`
